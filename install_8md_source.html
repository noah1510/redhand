1 # Manual installation
2 
3 This documentation should work on debian based linux distributions and on windows (I would include others but I am not familiar with them feel free to add them to the scripts).
4 
5 If you are on windows install [chocolatey](https://chocolatey.org/).
6 You can execute all commands and scripts in the powershell without the need for another shell, however you can use git bash, mingw, etc.
7 
8 * Install [git lfs](https://git-lfs.github.com/)
9 * Then clone the repository (`git clone https://github.com/noah1510/greenfoot-.git`)
10 * Then change into the directory of the project (`cd "greenfoot-"`)
11 * Run the build script with the init flag (`./scripts/build.sh --init`)
12 
13 If you want to you can build the testgame using the following command: `./testgame/scripts/execute.sh --init`
14 
15 Now you can continue with [customizing the build](https://github.com/noah1510/greenfoot-/wiki/Build-script), [contribute to further development]() or [start making your own game]().
